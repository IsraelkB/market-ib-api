# TWS API Tutorial in Python

In this tutorial, you will learn to use the IBKR TWS API in Python. By using the API, you will be able to analyze 
and backtest trading strategies. 

We will also code an example **Trading Bot** using TWS API.

## Coding environment
- Python 3.12
- Pycharm IDE
- Dependencies: Pandas, Plotly

## Section 1: API setup
TWS API Documentation: https://www.interactivebrokers.com/campus/ibkr-api-page/twsapi-doc/#api-introduction

1) TWS Trading Platform Download
   - Download link: https://www.interactivebrokers.com/en/trading/tws-offline-latest.php
2) Download & Install TWS API
   - Download link: https://interactivebrokers.github.io/#  
   - Windows installation: "cd ~/TWS API/source/pythonclient & python/python3 setup.py install  ""
   - You may need to install setuptools prior: "pip install setuptools"
3) Enable API Connection from TWS
4) Download & Configure Python Template Files
5) Connecting first Python script using TWS API
